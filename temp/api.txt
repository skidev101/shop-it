// lib/api/products.ts

```

**`store/` (Zustand)** — cart lives entirely in client state (persisted to localStorage). Auth token also lives here. Server state (products, orders) is handled by React Query or SWR.

**`middleware.ts`** — checks for auth token, redirects `/account/*` and `/checkout` to `/auth/login` if unauthenticated.

---

## Key Technical Decisions

| Concern | Recommended choice |
|---|---|
| Data fetching (server) | Native `fetch` in Server Components |
| Data fetching (client) | React Query (`@tanstack/react-query`) |
| Client state | Zustand |
| Forms | React Hook Form + Zod |
| Styling | Tailwind CSS |
| Payments | Stripe.js / Elements |
| Auth | JWT in httpOnly cookie via Express, or NextAuth |
| Images | `next/image` with your Express domain whitelisted |
```
